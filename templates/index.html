NYSERDA Solar Farms and Weather API

A REST API that joins New York solar farm generation data (from NYSERDA) with weather conditions (from Open-Meteo) and saves it as CSV.

Version 0

API Endpoints

GET /

This documentation page.

Example:
curl -X GET http://localhost:5000/
GET POST /searchFarms

Get a list of solar farms' name and coordinates within a certain capacity range and/or location range.

Parameters:
  • capacity_min (optional): Minimum capacity in kW
  • capacity_max (optional): Maximum capacity in kW
  • latitude, longitude, radius_km: Location-based search.
GET Example:
curl -X GET "http://localhost:5000/searchFarms?longitude=-73.7562&latitude=42.6526&radius_km=10"
GET POST /getFarm

Get data for one solar farm with or without weather data.

Parameters:
  • farm_name: Solar farm name (labeled by NYSERDA)
  • include_weather: Include weather data (true/false)
GET Example:
curl -X GET "http://localhost:5000/getFarm?farm_name=103 Sparling Road, LLC&include_weather=true"
GET POST /getFarms

Get data for multiple solar farms with or without weather data. Uses Haversine (Great Circle) distance to calculate coordinates to pull weather data.

Parameters:
  • farm_name: Solar farm name (labeled by NYSERDA)
  • include_weather: Include weather data (true/false)
GET Example:
curl -X GET "http://localhost:5000/getFarms?farm_name=103 Sparling Road, LLC;132 Pattersonville Rynex Corners Rd&include_weather=true"
